## Image Requirements

- Format: JPG or WebP (preferred)
- Optimization: Compressed for web
- Alt text: Descriptive and meaningful
- Dimensions: Maintain aspect ratios to prevent CLS
- Lazy loading: Enabled for below-fold images
- Width/Height attributes: Included to prevent layout shift
